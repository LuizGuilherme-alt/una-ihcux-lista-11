# ReCiclo - Protótipo de Baixa Fidelidade

## Integrantes

* Luiz Guilherme, Cauã Cordeiro, João Vitor Pedrosa, Silas Caleb

## Proposta de Valor

O aplicativo ReCiclo tem como objetivo conectar pessoas que desejam doar itens com outras que precisam, promovendo sustentabilidade, reutilização e redução de desperdício.

## Descrição das Telas

**1. Home / Feed**
Exibe os itens disponíveis próximos ao usuário, com imagem, nome e distância.

**2. Filtros**
Permite refinar a busca por categoria, distância e estado de conservação.

**3. Detalhes do Item**
Mostra informações completas do item e permite entrar em contato com o doador.

**4. Cadastro de Item**
Tela onde o usuário pode cadastrar um novo item para doação.

**5. Chat**
Permite a comunicação entre doador e interessado para combinar a retirada.

**6. Perfil**
Exibe informações do usuário e suas conquistas, como o Selo Eco.

## Jornada do Usuário

O usuário acessa o aplicativo → navega pelos itens disponíveis → seleciona um item → entra em contato com o doador → combina a retirada → conclui a doação → recebe um selo como recompensa.
